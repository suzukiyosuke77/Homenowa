## ホメノワ
※漢字だと「褒めの輪」と書きます。

## サイト概要
会員制エンタメ総合レビューサイト。
想定ジャンルは映画、演劇、舞台、プロレスなど。
ただしネガティブなレビューは書き込めず、ポジティブな（褒める）レビューしか投稿できない。
マイナス意見の多いレビューや、ポジティブを装った棘のあるレビューが投稿されるのを防ぐため、管理人がまず投稿されたレビューをチェックする。
問題がなければそこで初めてウェブ上に公開される。
自分のアツい思いを吐き出すため「だけ」の場所であり、公開されたレビューに第三者が具体的なコメントを残す事はできない（ただし『いいね』ボタンだけは実装）。

### サイトテーマ
エンターテイメント体験の感動を投稿できるレビューサイト。

### テーマを選んだ理由

理由はズバリ「好きなジャンルはポジティブな意見だけ目にしたい！」と言う私の願望です。

そもそも私は、上記の例で挙げた演劇、映画、ライブ、プロレスなど、様々なエンターテインメントが好きで、他の人がどのように感じ、
感動しているのかを知りたくて、SNSなどでレビューを探します。しかし、そこで目にするのは必ずしもポジティブな意見ばかりではありません。
中には、明らかに悪意を感じるレビューや感情的に攻撃する内容などもあり、気分が沈んでしまうこともあります。
そのようにネガティブなレビューが多いと、感動が色褪せてしまったり、自分の価値観が間違っているのではないか？と感じてしまうこともあります。
エンタメ作品は様々な感じ方があって当然ですが、自分の好きなものについてのネガティブな意見ばかりが目に入ると、どうしてもそうした不安に駆られてしまいます。

こうした経験から、私は「自分の好きな『推し』やジャンルにはポジティブな意見だけ集まって欲しい」という思いを持つ人が、
私以外にもたくさんいるのではないかと考えるようになりました。
特にプロレスというジャンルは、私自身が大好きなのですが、どうしてもニッチなジャンルです。
熱烈なファンがいる一方で、一般的にはまだ広く知られていない部分があるのです。

ですがこのサービスが活性化すれば、ポジティブな意見だけが集まるため、
レビューを見た人も「観てみたい！」「そんなに面白いなら体験してみたい！」と言った前向きな気持ちが刺激され、
ジャンルの新たなファン層の拡大にも繋がるはずです。

このサービスは単にポジティブなレビューを共有する場、と言うだけではありません。
エンタメの魅力、自分の感動を広く伝えていくことで、エンタメ自体が持った楽しさを最大限に引き出すことが目的です。
ポジティブなレビューの力で、新たなファン層を獲得し、ジャンル自体の活性化を狙っています。

### ターゲットユーザ
・エンタメが好きで、自分の感動をみんなに広めたいと考えている人
・多種多様な意見が氾濫しているSNSに疲れた人
・ネガティブな意見は悲しくなるから目にしたくない、単純でも誰かのポジティブな意見だけ見たい、と思っている人​

### 主な利用シーン
・エンターテイメント体験における感動を他者と共有したい時
・他の人が同じエンターテイメントでどんな風に感動したか知りたい時
・ポジティブなレビューの集まるエンタメを知りたい時
