アプリ名：ホメノワ(褒めの輪)

## サイト概要
会員制エンタメ総合レビューサイト。
想定ジャンルは映画、演劇、舞台、プロレスなど。
ただしネガティブなレビューは書き込めず、ポジティブな（褒める）レビューしか投稿できない。
マイナス意見の多いレビューや、ポジティブを装った棘のあるレビューが投稿されるのを防ぐため、管理人がまず投稿されたレビューをチェックする。
問題がなければそこで初めてウェブ上に公開される。
自分のアツい思いを吐き出すため「だけ」の場所であり、公開されたレビューに第三者が具体的なコメントを残す事はできない（ただし『いいね』ボタンだけは実装）。

### サイトテーマ
エンターテイメント体験の感動を投稿できるレビューサイト。

### テーマを選んだ理由

私は演劇、映画、ライブ、プロレスなど、様々なエンターテインメントが好きで、他の人がどのように感じ、感動しているのかを知りたくて、
SNSなどでレビューを探します。しかし、そこで目にするのは必ずしもポジティブな意見ばかりではありません。
中には、明らかに悪意を感じるレビューや感情的に攻撃する内容もあり、気分が沈むことがあります。

そのようにネガティブなレビューが多いと、感動が色褪せたり、自分の価値観が間違っているのではないかと感じてしまうこともあります。
エンタメ作品は様々な感じ方があって当然ですが、自分の好きなものについてのネガティブな意見ばかりが目に入ると、
どうしてもそうした不安に駆られてしまいます。

こうした経験から、私は「自分の好きな『推し』やジャンルにはポジティブな意見だけ集まって欲しい」という思いを持つ人が、
他にもいるのではないかと考えるようになりました。
特にプロレスというジャンルは、私自身が大好きなのですが、どうしてもニッチなジャンルです。
濃いファン同士での話題にとどまり、一般的に広まることがありません。
ですが、もしポジティブな意見が増えていけば、「そんなに面白いなら観てみようか」と自然に新たな興味を持つ人が現れ、
もっと広く認知されるようになるはずです。

このように、「自分の好きなジャンルや作品がもっと評価されてほしい」と感じる人が、
そのアツさを共有できる場所を提供したいと考え、制作してみようと考えました。
​
### ターゲットユーザ
・エンタメが好きで、自分の感動をみんなに広めたいと考えている人
・多種多様な意見が氾濫しているSNSに疲れた人
・ネガティブな意見は悲しくなるから目にしたくない、単純でも誰かのポジティブな意見だけ見たい、と思っている人​

### 主な利用シーン
・エンターテイメント体験における感動を他者と共有したい時
・他の人が同じエンターテイメントでどんな風に感動したか知りたい時
・ポジティブなレビューの集まるエンタメを知りたい時
